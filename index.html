Просмотр изменений и создание коммитов (фиксация изменений) $ git status Перечисляет все новые или изменённые файлы, которые нуждаются в фиксации $ git diff Показывает различия по внесённым изменениям в ещё не проиндексированных файлах $ git add [файл] Индексирует указанный файл для последующего коммита $ git diff --staged Показывает различия между проиндексированной и последней зафиксированной версиями файлов $ git reset [файл] Отменяет индексацию указанного файла, при этом сохраняет его содержимое $ git commit -m "[сообщение с описанием]" Фиксирует проиндексированные изменения и сохраняет их в историю версий
